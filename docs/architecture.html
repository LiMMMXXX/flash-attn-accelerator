◆ 标准 Softmax 的问题

标准注意力中，计算 O = softmax(QKᵀ)V 需要将完整的 S = QKᵀ (N×N 矩阵) 写入 HBM，然后逐行计算 softmax，再读回来做矩阵乘法。这导致 O(N²) 的 HBM 访问。

◆ Flash 的核心洞察

软最大值可以增量计算：

  • 维护运行中统计量：m = running max, ℓ = running sum of exp
  • 处理完每个分块 (Kⱼ, Vⱼ) 后，对之前的输出按比例重缩放 (rescaling)
  • 最终结果与标准 softmax 精确一致 (bitwise identical)
